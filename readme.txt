0. Requirements: Python 3.8
1. load the dependencies: "pip install -r requirements.txt"
2. Training takes a long time, use "python train-evaluate-main.py -loadModel" to load the saved model.
   The model can be downloaded via: https://drive.google.com/file/d/12Cc8CMMx7ejPBKNdP8yPuTqRLno75zUk/view?usp=sharing
   Unzip in root.
3. Running "python train-evaluate-main.py -trainModel" will train, save and evaluate the model
4. Run "python test.py" to see a prediction of a randomly picked sentence
